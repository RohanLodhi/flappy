# Flappy
Flappy bird pygame 

# Features
* Uses pygame

* Open Source

## Clone with HTTPS
```
git clone https://github.com/RohanLodhi/flappy.git
```

## Clone with SSH
```
git clone git@github.com:RohanLodhi/flappy.git
```

## Install pip3 to use the script
```
sudo apt install pip3 
```

## Install requests and bs4
```
pip3 install pygame
```

## Run the Script
```
python3 game.py
```

## Enjoy!
